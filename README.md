### 三、功能描述
系统分为（身份）：普通用户、物业、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 个人中心（修改头像、修改个人信息、修改密码）
4. 我的报修
5. 我的车位
6. 我的房屋
7. 物业费缴纳
8. 我的私信
9. 首页
10. 公告一览
11. 物业人员投诉
12. 物业人员一览

**物业功能：**
1. 登录
2. 注册
3. 首页
4. 个人中心（修改头像、修改个人信息、修改密码）
5. 用户管理
6. 报修管理
7. 车位管理
8. 出入管理
9. 房屋管理
10. 物业费缴纳管理
11. 公告管理
12. 无业人员投诉管理
13. 我的私信管理

**管理员功能：**
1. 登录
2. 首页
3. 个人中心（修改头像、修改个人信息、修改密码）
4. 管理员管理
5. 物业人员管理
6. 用户管理
7. 报修管理
8. 车位管理
9. 出入管理
10. 房屋管理
11. 物业费缴纳管理
12. 公告管理
13. 无业人员投诉管理
14. 我的私信管理
15. 基础数据管理（报修类型、报修状态、车位类型、出入类型、单元、楼号、缴费类型、缴费状态、公告类型、人员类型）
16. 轮播图管理
